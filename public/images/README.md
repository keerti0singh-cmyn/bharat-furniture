# Images Directory

This directory contains images for the Bharat Furniture website.

## Instructions for Adding Images:

1. Add your furniture product images in this directory
2. Use descriptive filenames (e.g., `bed-king-size.jpg`, `soa-3-seater.jpg`)
3. Recommended image sizes:
   - Hero banners: 1920x1080px
   - Product images: 800x600px
   - Thumbnails: 400x300px
4. Use WebP or JPEG format for optimal performance
5. Compress images before uploading

- Example: `dining-table-6-seater.jpg`
